Objetivo do Projeto

Este projeto tem como objetivo desenvolver um novo site para a ACB (Associação de Conselheiros do Brasil), utilizando como referência o site atual: acbrasil.org.br.

Visão do Projeto
	
    1.	A ACB Brasil é uma organização voltada a apoiar empresas brasileiras na adoção de práticas de governança corporativa.

	2.	Empresas que crescem sem adotar um modelo de gestão eficiente tendem a enfrentar crises e desorganização. Por isso, a ACB Brasil reúne uma equipe de profissionais com mais de 20 anos de experiência em liderança operacional, preparada para orientar empresas no caminho de uma gestão mais estruturada e sustentável.

	3.	O site deverá contar com uma seção de blog, na qual novas notícias e conteúdos poderão ser publicados ao longo do tempo por colaboradores da ACB.
    3.1. Os usuários deverão poder se cadastrar na newsletter e também manifestar interesse em se tornarem associados.

	4.	Resumos dos artigos deverão ser exibidos na página inicial. Ao serem clicados, deverão direcionar o usuário ao artigo completo
